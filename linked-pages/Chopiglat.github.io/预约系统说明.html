预约关注系统使用说明

系统优化完成

预约关注系统已全面优化，解决了之前"coming-soon.html"页面无实际功能的问题。

主要改进：

  • 从简单的占位页面升级为完整的预约系统
  • 新增表单验证和实时反馈
  • 支持数据存储和管理
  • 提供多种使用方案（PHP/纯JS）
  • 响应式设计，适配所有设备

系统文件说明

主要文件：

  • subscribe.html - 完整预约页面（推荐）
  • subscribe-js.html - 纯JavaScript版本（无需服务器）
  • process-subscribe.php - PHP后端处理脚本
  • admin.html - 预约管理后台
  • coming-soon.html - 原占位页面（已废弃）

使用方案

方案1：纯前端版本（推荐测试用）

  • 使用 subscribe-js.html
  • 数据存储在浏览器本地
  • 无需服务器支持
  • 通过 admin.html 查看数据

方案2：PHP版本（推荐生产用）

  • 使用 subscribe.html
  • 数据存储在服务器文件
  • 需要PHP环境支持
  • 支持CSV导出

链接更新

所有"预约关注"按钮已更新为指向新的预约系统：

  • 首页预约按钮 → subscribe.html
  • 作品集预约按钮 → ../subscribe.html
  • 猪鼓励艺术实验室预约 → subscribe.html
